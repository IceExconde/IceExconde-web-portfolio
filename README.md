# Portfolio

A simple portfolio website I made.

## Sections

- **Home**: Hero section with introduction
- **About**: Personal information and background
- **Projects**: Showcase of my work
- **Contact**: Contact form for email communication

## Getting Started

Simply open `index.html` in your web browser to view the portfolio.

## Customization

- Update social media links in the navbar and footer (lines with `href="https://github.com"`, etc.)
- Modify the contact form email address in `script.js` (replace `your-email@example.com`)
- Edit project cards in the Projects section
- Customize colors in `styles.css` using CSS variables

## Technologies Used

- HTML5
- CSS3 (with CSS Variables)
- Vanilla JavaScript
